# CampusHubSystem

CampusHubSystem is a College Management System built using the MERN Stack.

## Tech Stack
- React.js (Frontend)
- Node.js
- Express.js
- MongoDB

## Features
- Student Management
- Teacher Management
- Attendance System
- Notice Board
- Result Management
- Admin Dashboard

## Project Structure
CampusHubSystem
 ├── frontend (React App)
 └── Backend (Node + Express API)

## Installation

### Backend
cd Backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
